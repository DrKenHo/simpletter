こんにちは！

このページが見えているということは、あなたの環境は正常に動作しています。
サンプルを動かしてみても良いかもしれません。

ワークショップでやること

このワークショップは以下のステップで行います。
  • ハローワールド！
  • ハローワールドを作ってみよう！
  • APIを呼んでみよう！
  • タイムラインを表示しよう！
  • つぶやこう！

ハローワールド！

このページを表示……できてますね！おめでとうございます！！

ハローワールドを作ってみよう！

決まった内容を表示するだけのシンプルなページを作成します。 HTMLで作成した画面に、あなたが設定した値を表示してみましょう。

ここで学べること

  • Thymeleaf を使用したシンプルなページの作り方
  • Spring MVC を使用したシンプルなControllerの作り方
  • Spring を使用した設定値の取得方法

APIを呼んでみよう！

簡単なAPI呼び出しを行います。 サーバーから取得した値を、画面に表示してみましょう。

ここで学べること

  • RestTemplate を使用したAPIの呼び出し方

タイムラインを表示しよう！

いよいよタイムラインを取得します。 APIの呼び出し方は先ほどと同じですが、今回はAPIの結果をあなたが作るクラスのインスタンスに変換します。 また、タイムラインには複数件のツイートが入っています。

ここで学べること

  • JSONから自分で作成したクラスへのマッピング
  • Thymeleafで複数件を扱う方法
  • Thymeleafでプロパティを扱う方法

つぶやこう！

入力した言葉をつぶやけるようにしましょう。 API呼び出し時に、入力したメッセージも渡せばOKです。 この時、APIが受け取れる形で渡してあげなくてはいけません。

ここで学べること

  • フォームの入力内容をControllerで受け取る方法
  • APIにパラメータを渡す方法
  • リダイレクト

設定情報

この設定は状況に合わせて変更してください。
"API Server URL" は、用意されているアドレスを設定します。
"User ID" は、あなたがこのワークショップで使用するユーザーIDを設定します。API Serverで登録したUser IDを指定してください。
